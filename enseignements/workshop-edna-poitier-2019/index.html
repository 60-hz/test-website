Concepts du logiciel Zapworks Studio

Zapworks Studio  est un logiciel qui permet de designer rapidement des experiences de réalité  augmentée et de réalité mixte.  Il permet d'intégrer et de scénariser des images, des videos, des modèles animés en 3d et  des sons.

 

Zappar est le lecteur  iOS et Android permettant de lire les expériences de réalité augmenté réalisées avec Zapworks studio.

 

Il est également possible de lire la réalité augmenté directement dans le navigateur web ( webAR: plus lent mais plus accessible ), grace à un QRcode généré à partir de l'interface web ( accessible depuis votre compte ).

Démarrage avec Zapworks Studio

 

1 - Créez un compte personnel “Hobby Account” ici https://zap.works/pricing/personal/ (utilisez le login avec Facebook ou google ) puis cochez la case “I am using zapworks for personal use only and not commercial”

2 - Téléchargez Zapworks Studio pour WINDOWS or APPLE

3 - Installez-le, Loguez vous, Vous êtes prêt!

-> N'oubliez pas d'activer votre compte dans le mail reçu pour publier.

 

 


Aspects techniques  & conseils

 

- Garder un projet < 15 mo (téléchargement rapide).

- Target image: .jpg ou  .png, Maximum 2048*2048 pixels conseillé. Mode couleur RGB (pas de CMJN). Images contrastées, éviter logo minimals.

- Video: .mp4, (H264 codec), de préférence hébergé en ligne (ex: https://videozapworks.s3.eu-west-3.amazonaws.com/video1.mp4).

- Sons: .mp4, (AAC codec), 160kbps, 44100hz.

- Modèles 3d supportés: fbx ,  obj + mtl ,   pod,  dae   (low poly, max 10 mo / modèle conseillé. Rig Animation, maximum 256 bones/mesh).

- Triggers: Zappar / webAR https://docs.zap.works/general/project-triggers/
